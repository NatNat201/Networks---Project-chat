Premièrement vous devez mettre tous les fichiers py dans le même dossier

LANCEMENT DU SERVEUR:
Lancer l'invite de commande
Rentrer le chemin d'accès vers le dossier où sont situés les .py
Taper python/py/python3 (suivant la version intaller sur votre ordinateur) server.py host (localhost/IP)
Le numéro de port est défini par défaut donc il est inutile de le préciser
Press enter

LANCEMENT D'UN CLIENT:
Lancer l'invite de commande
Rentrer le chemin d'accès vers le dossier où sont situés les .py
Taper python/py/python3 (suivant la version intaller sur votre ordinateur) client.py host (localhost/IP)
Le numéro de port est défini par défaut donc il est inutile de le préciser
Press enter

NOTE: 
De plus tout fichier susceptible d'être envoyé doit également se trouver dans ce dossier
